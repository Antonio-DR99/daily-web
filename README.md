# Web Practice - Daily Content

Práctica de desarrollo web con PHP, JavaScript y CSS, gestionando contenido diario, galería de imágenes y reproductor de música.

## Funcionalidades

- Mostrar noticias diarias según el día seleccionado.
- Galería de imágenes con modal interactivo.
- Reproductor de música con selección de canciones aleatorias.
- Validación de entradas del usuario.
- Experiencia tipo panel estilo retro (Windows 95).

## Tecnologías utilizadas

- **PHP**: Backend y manejo de archivos dinámicos.
- **HTML & CSS**: Estructura y estilo de las páginas.
- **JavaScript**: Interactividad en galería y reproductor.
- **Glob y gestión de directorios**: Organización de imágenes, música y noticias.
